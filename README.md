## Về dự án

Website này được tạo để gửi lời mời tham dự lễ thành hôn đến gia đình và bạn bè.

## Công nghệ sử dụng

- **React 18** - Thư viện JavaScript để xây dựng giao diện người dùng
- **TypeScript** - Ngôn ngữ lập trình có kiểu tĩnh
- **Vite** - Build tool hiện đại và nhanh
- **Tailwind CSS** - Framework CSS utility-first
- **Shadcn/UI** - Bộ component UI với Radix UI
- **React Router** - Điều hướng trang
- **React Hook Form** - Quản lý form
- **TanStack Query** - Quản lý trạng thái server
- **Vitest** - Framework testing
