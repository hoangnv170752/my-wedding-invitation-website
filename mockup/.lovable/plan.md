
# Wedding Invitation Website

A beautiful, elegant single-page wedding invitation website inspired by the Lavelo template, with a white and soft pastel color scheme. Fully responsive across all screen sizes.

## Design Style
- **Clean white** background with soft gold/rose-gold accent colors
- Elegant serif fonts for headings, clean sans-serif for body text
- Subtle animations on scroll (fade-in, slide-up)
- Floral/decorative dividers between sections
- Placeholder images with romantic styling

## Sections (Top to Bottom)

### 1. Navigation Bar
- Sticky top navigation with smooth scroll links to each section
- Hamburger menu on mobile
- Semi-transparent background that becomes solid on scroll

### 2. Hero Section
- Full-screen hero with a romantic background image overlay
- Couple's names prominently displayed with elegant typography
- Wedding date and "Save the Date" text
- Scroll-down indicator animation

### 3. Countdown Timer
- Live countdown to the wedding date (days, hours, minutes, seconds)
- Beautifully styled countdown boxes with decorative borders

### 4. Couple Section
- Side-by-side photos of the bride and groom (placeholders)
- Names, brief bios, and social links
- Decorative dividers and elegant layout

### 5. Love Story / Timeline
- Vertical timeline of the couple's milestones (first meeting, first date, proposal, etc.)
- Alternating left/right layout on desktop, single column on mobile
- Placeholder dates and descriptions

### 6. Events Section
- Cards for ceremony and reception details
- Date, time, and venue information
- Map placeholder or venue address
- Dress code information

### 7. Gallery Section
- Photo grid with lightbox functionality
- Responsive masonry or grid layout
- Placeholder romantic images

### 8. Wedding Party
- Grid of bridesmaids and groomsmen
- Photo, name, and role for each
- Responsive cards layout

### 10. Footer

## Responsiveness
- Fully responsive from 320px mobile to 1920px+ desktop
- Mobile-first approach with breakpoints for tablet and desktop
- Touch-friendly navigation and interactions
- Optimized spacing and typography scaling for all screen sizes
